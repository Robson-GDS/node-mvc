# node-mvc

## Sobre o projeto

Projeto desenvolvido para entender e fixar sobre MVC, ferramenta para criar, editar e excluir tasks.

## Tecnologias

- NodeJS
- express
- express-handlebars
- mysql2
- nodemon
- sequelize

## Instalação e uso

```
# Clone o repositório
git clone https://github.com/Robson-GDS/node-mvc.git

# Acesse a pasta do projeto
cd node-mvc

# Instale as dependências
npm install

# Execute a aplicação
npm run dev

```
